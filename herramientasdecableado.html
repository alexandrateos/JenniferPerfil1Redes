Seccion de herramientas para cableado

Punto cinco de la actividad.

Herramientas utilizadas para realizar cableado estructurado, agregar imagen y descripcion tecnica de cada una

Ponchadora

Una ponchadora de cables, también conocida como crimpadora o crimpadora de terminales, es una herramienta hidráulica o manual que utiliza presión para unir conectores o terminales a los extremos de cables eléctricos.

Crimpadora

Una crimpadora es una herramienta especializada que se utiliza para crimpar un conector o terminal a un cable. Este proceso implica deformar el conector para que encaje bien en el cable. Esto garantiza una conexión mecánica sólida y una conexión eléctrica confiable.

Desforradora

Una desforradora es una herramienta diseñada para quitar el aislamiento o el recubrimiento de un cable eléctrico sin dañar sus conductores internos. Este proceso es crucial para preparar los cables para soldar, crimparlos o conectarlos a los terminales. Las desforradoras son esenciales para instalaciones eléctricas, electrónicas, de telecomunicaciones y industriales.

Tester

Un testador, también conocido como multímetro, es un instrumento de medición eléctrica que se utiliza para inspeccionar y diagnosticar circuitos eléctricos. Es capaz de medir una variedad de parámetros eléctricos básicos, incluidos voltaje (V), corriente (A), resistencia (Ω), y ocasionalmente, continuidad, capacitancia, frecuencia y temperatura.
